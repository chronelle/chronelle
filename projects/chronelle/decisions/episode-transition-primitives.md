# Episode and Transition Primitives

Chronelle will model meaningful organizational activity as Episodes.

Chronelle will model organizational state change as Transitions.

Episodes group related activity into human-legible memory.

Transitions record exact changes to Goals, Constraints, Assumptions, Questions,
Alternatives, Decisions, and other future ontology primitives.

Current project state should be derivable from Transitions and readable through
Projections.
