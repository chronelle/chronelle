# Use Task Planning Projection

Chronelle will start planning work with a Task Planning projection.

Tasks will not become ontology primitives yet.

Tasks are proposed next actions derived from project memory.

Episodes and Transitions remain the durable memory of what happened and what
changed.

This lets Chronelle plan work now while preserving the option to promote Task to
an ontology primitive later.
