# Create Ontology Change Consensus Constraint

## Rationale

The decision to express consensus requirements as Constraints needed a concrete
example, and Chronelle's own ontology changes are a real case.

## Before

Chronelle had no recorded consensus requirement.

## After

Chronelle records a Constraint that material ontology changes require human
steward agreement, which the Consensus projection reads as the membership and
quorum for ontology-change targets.
