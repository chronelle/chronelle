# Create Actor Ontology

## Rationale

The Actor primitive needed a shared definition, and the ontology index needed to
list it.

## Before

Chronelle had no ontology document for participants.

## After

Chronelle has `ontology/actor.md` defining what an Actor records, how it is
referenced, and its lifecycle, and the ontology README lists Actor among the
durable primitives.

## Evidence

- `ontology/actor.md`
- `ontology/README.md`
