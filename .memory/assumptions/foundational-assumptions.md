# Foundational Assumptions

## Shared Organizational Memory

Organizations require memory that both humans and agents can inhabit.

## Intent Drifts From Execution

Execution naturally diverges from intent unless synchronization occurs.

## Decisions Are More Valuable Than Tasks

Long-term organizational memory depends on preserving decisions and rationale.

## Organizational Context Is Graph Structured

Organizational knowledge is fundamentally relationship-oriented rather than document-oriented.
