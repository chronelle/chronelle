# Ontology Changes Require Steward Consensus

Material changes to the Chronelle ontology require agreement from the human
steward Actor before they are treated as ratified.

This is an example of a consensus requirement expressed as a Constraint: it
defines, for ontology-change targets, who must agree.

The Consensus projection reads this Constraint to decide whether an ontology
Decision is `agreed` or only `proposed`.
